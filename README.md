 ⚙️ Smart Expense Tracker - Backend

This is the backend service for the **Smart Expense Tracker** application. It handles all business logic, APIs, and data management for tracking expenses and income.

---

## 🚀 Features

- 📡 RESTful API for expense management  
- ➕ Add, update, and delete transactions  
- 📂 Categorize expenses and income  
- ⚖️ Calculate total balance  
- 🧾 Store transaction history  
- 🔄 Real-time data handling (if integrated)  

---

## 🛠️ Tech Stack

**Backend:**
- Node.js  
- Express.js  

**Database:**
- MongoDB  

**Tools:**
- Mongoose  
- dotenv
